Tcpdump


In this project I introduced tcpdump and some if its features. Tcpdump is a the premier network analysis tool for information security and networking professionals. As an IT Support Specialist, having a solid grasp of this application is essential in understanding TCP/IP. Tcpdump helps display network traffic in a way that's easier to analyze and troubleshoot.

Overview


  • Command basics- Usage of tcpdump and what some of its flags do, as well as interpret the output.
  • Packet captures- Practice saving packet captures to files and reading them back
  • Write to a file in CSV format
  • Move files to the appropriate directory for use with the CSV --> HTML converter
